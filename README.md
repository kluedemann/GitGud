## Running the game
 1. `python3 -venv env`
 2. `source env/bin/activate`
 3. `pip install -r requirements.txt`
 4. `python main.py`

## Compiling the game
  1. `pip install pyinstaller`
  2. `pyinstaller --onefile main.py`
  3. `.\dist\main`
